### TclText

![Simulators Structure](SimulatorsView.svg)

#### Purpose

Internal static helper class that quotes arbitrary string values (VHDL file paths, test-bench entity
names) for safe interpolation into the TCL do-scripts generated by `ModelSimSimulator`,
`QuestaSimSimulator`, and `ActiveHdlSimulator`. Ensures values containing spaces or TCL metacharacters
(`$`, `[`, `]`, `"`, `\`) are treated as literal text rather than being substituted or split by the
TCL interpreter.

#### Data Model

No instance or static state; `TclText` is a stateless static class exposing a single pure function.

#### Key Methods

**Quote**: Returns a TCL-quoted representation of a string value.

- *Parameters*: `string value` — the value to quote. Must not be null.
- *Returns*: `string` — a TCL-quoted representation of `value`.
- *Preconditions*: None.
- *Postconditions*: Returns `"{" + value + "}"` unchanged when `value` contains no literal `{` or `}`
  characters (brace-quoting suppresses all TCL substitution). Otherwise returns a double-quoted form
  with each of `\`, `"`, `$`, `[`, `]`, `{`, and `}` individually backslash-escaped, escaping the
  backslash first so the escaping backslashes themselves are not re-escaped.

#### Error Handling

`Quote` throws `ArgumentNullException` when `value` is null; it has no other failure modes.

#### Dependencies

- **System.Text.StringBuilder** — builds the escaped double-quoted fallback form.

#### Callers

- **ModelSimSimulator** — quotes source file paths and test names before interpolating them into
  generated `.do` scripts.
- **QuestaSimSimulator** — quotes source file paths and test names before interpolating them into
  generated `.do` scripts.
- **ActiveHdlSimulator** — quotes source file paths and test names before interpolating them into
  generated `.do` scripts.
