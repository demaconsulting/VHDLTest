### XilinxArgText

![Simulators Structure](SimulatorsView.svg)

#### Purpose

Internal static helper class that quotes arbitrary string values (VHDL file paths, test-bench entity
names) for safe interpolation into the Xilinx argument files (`-file`/`-f` inputs to `xvhdl`/`xelab`)
generated by `VivadoSimulator`. Xilinx argument files are not TCL scripts; per Xilinx UG900, they use
standard shell-style tokenization where a value containing whitespace or other special characters must
be double-quote wrapped, with embedded `"` and `\` characters backslash-escaped.

#### Data Model

No instance or static state; `XilinxArgText` is a stateless static class exposing a single pure function.

#### Key Methods

**Quote**: Returns a double-quoted representation of a string value for a Xilinx argument file.

- *Parameters*: `string value` — the value to quote. Must not be null.
- *Returns*: `string` — a double-quoted representation of `value`.
- *Preconditions*: None.
- *Postconditions*: Always wraps `value` in double quotes, with each embedded `\` and `"` character
  individually backslash-escaped, escaping the backslash first so the escaping backslashes themselves
  are not re-escaped.

#### Error Handling

`Quote` throws `ArgumentNullException` when `value` is null; it has no other failure modes.

#### Dependencies

- **System.Text.StringBuilder** — builds the escaped double-quoted form.

#### Callers

- **VivadoSimulator** — quotes source file paths and test names before interpolating them into
  generated `.do` argument files.
