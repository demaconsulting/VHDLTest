# Introduction

This document records the static analysis results for VHDLTest.

## Purpose

To provide evidence that VHDLTest has been analyzed for code quality issues and that any findings
have been reviewed and resolved or accepted.

## Scope

Covers static analysis of all source code in `src/` for VHDLTest. Test code is excluded from
static analysis requirements.

The analysis includes:

- CodeQL static analysis results (security vulnerabilities and code quality findings)
- SonarCloud quality gate status, code issues, security hotspots, technical debt, and code
  coverage metrics
